1. Árvore do documento (DOM) e seletores avançados

1.1 O que é o DOM?

O Document Object Model (DOM) é uma representação hierárquica (em árvore) dos elementos HTML. Entender essa hierarquia permite aplicar estilos de forma precisa, usando relações como pai-filho, ancestral-descendente e irmãos. No CSS, podemos mirar elementos com base nessas relações, sem precisar de classes ou IDs.

📌 Exemplo prático de hierarquia:
div > p → seleciona apenas parágrafos que são filhos diretos de uma div.
div p → seleciona todos os parágrafos descendentes da div (qualquer nível).
h1 + p → seleciona o parágrafo que vem imediatamente após um h1 (irmão adjacente).
h1 ~ h2 → seleciona todos os h2 que são irmãos posteriores a um h1.

1.2 Seletores estruturais (first-child, last-child, etc.)

Esses seletores permitem estilizar elementos com base na posição dentro do pai.

  • li:first-child → primeiro item de uma lista.
  • li:last-child → último item.
  • p:nth-child(2) → segundo filho do tipo parágrafo (independente do tipo).
/* Exemplo: estiliza o primeiro link dentro de um nav */
nav > a:first-child {
    font-weight: bold;
    color: #A67E6F;
}

1.3 Herança no CSS

Algumas propriedades (como color, font-family) são herdadas pelos elementos filhos. Outras (como width, border) não são. Saber isso evita repetição de código.

2. Pseudoclasses e pseudoelementos

2.1 Pseudoclasses: estilizando estados

Pseudoclasses (:) definem o estilo de um elemento em um estado específico.

PseudoclasseAplicaçãoExemplo real
:linkLink não visitadoa:link { color: #243E73; }
:visitedLink já visitadoa:visited { color: #A67E6F; }
:hoverMouse sobre o elementoMenu que muda de cor ao passar o mouse
:activeElemento sendo clicadoBotão com efeito de clique
:focusCampo de formulário selecionadoinput:focus { border: 2px solid #8BB4D9; }
:checkedRadio ou checkbox marcadoPersonalizar botões de opção
:enabled / :disabledCampos habilitados/desabilitadosEstilizar campos bloqueados com fundo cinza

Ordem importante para links: :link → :visited → :hover → :active (LVHA).

2.2 Pseudoelementos: estilizando partes específicas

Pseudoelementos (::) permitem estilizar partes de um elemento ou inserir conteúdo virtual.

  • ::first-letter → primeira letra (cria efeito de letra capitular).
  • ::first-line → primeira linha de um texto.
  • ::before e ::after → inserem conteúdo antes/depois do elemento (ex: ícones, aspas).
  • ::marker → estiliza marcadores de lista (bullets ou números).
  • ::selection → estiliza o texto selecionado pelo usuário.
/* Exemplo prático: letra capitular */
p::first-letter {
    font-size: 40px;
    font-weight: bold;
    color: #A67E6F;
    float: left;
    margin-right: 8px;
}

/* Inserir ícone antes de cada link externo */
a[target="_blank"]::after {
    content: "🔗";
    margin-left: 5px;
}

3. Barra de navegação (menus) e botões

3.1 Menu vertical estilizado

Menus são tradicionalmente construídos com listas <ul>. Para um menu vertical profissional:

ul {
    list-style-type: none;
    margin: 0;
    padding: 0;
    width: 200px;
    background-color: #8BB4D9;
}
li a {
    display: block;  /* torna toda a área clicável */
    color: #1E3259;
    padding: 12px 16px;
    text-decoration: none;
}
li a:hover {
    background-color: #243E73;
    color: #F2D8CE;
}

3.2 Menu horizontal com display: inline-block ou float

Para transformar o menu vertical em horizontal, basta mudar os <li> para display: inline-block; ou usar float: left;.

li {
    display: inline-block;
}
/* ou */
li {
    float: left;
}

3.3 Menu suspenso (dropdown) com subnível

Um submenu é criado aninhando outra <ul> dentro de um <li>. No CSS, usamos position: relative no item pai e position: absolute no submenu, que fica oculto (display: none) até o hover.

<ul>
  <li><a href="#">Home</a>
    <ul>
      <li><a href="#">Sobre</a></li>
      <li><a href="#">Lojas</a></li>
    </ul>
  </li>
  <li><a href="#">Notícias</a></li>
</ul>
li { position: relative; }
li ul {
    position: absolute;
    left: 150px;
    top: 0;
    display: none;
}
li:hover ul {
    display: block;
}

3.4 Botões estilizados

Botões podem ser criados com <button>, <a> ou <input>. Aplique uma classe .btn para padronizar.

.btn {
    background-color: #243E73;
    border: none;
    color: white;
    padding: 12px 24px;
    text-align: center;
    text-decoration: none;
    display: inline-block;
    font-size: 16px;
    border-radius: 30px;
    transition: background 0.3s;
}
.btn:hover {
    background-color: #8BB4D9;
    cursor: pointer;
}

4. Dimensionamento, variáveis e formatação de imagens

4.1 Box-sizing: controlando o cálculo de largura

Por padrão, width + padding + border = largura real. Para incluir padding e border dentro da largura definida, use box-sizing: border-box. Recomenda-se aplicar a todos os elementos:

* {
    box-sizing: border-box;
}

4.2 Variáveis CSS (Custom Properties)

Variáveis evitam repetição e facilitam manutenção. Defina no :root (escopo global) e use com var().

:root {
    --cor-primaria: #243E73;
    --cor-destaque: #A67E6F;
    --espaco-padrao: 16px;
}
h1 {
    color: var(--cor-primaria);
    margin-bottom: var(--espaco-padrao);
}

4.3 Formatações para imagens

  • Cantos arredondados: border-radius: 10px;
  • Imagem circular: border-radius: 50%; (com largura/altura iguais)
  • Opacidade: opacity: 0.8;
  • Centralizar: display: block; margin: auto;
  • Responsiva: max-width: 100%; height: auto;

5. Movimentos em CSS: transformações 2D e transições

5.1 Transformações 2D (transform)

A propriedade transform permite mover, girar, escalar e inclinar elementos.

MétodoEfeitoExemplo
translate(x, y)Move o elementotransform: translate(50px, 20px);
rotate(deg)Gira o elementotransform: rotate(45deg);
scale(x, y)Altera o tamanhotransform: scale(1.5); (aumenta 50%)
skewX(deg) / skewY(deg)Inclina o elementotransform: skewX(10deg);

5.2 Transições suaves

Transições (transition) criam animações suaves entre estados. Defina a propriedade e a duração.

.box {
    width: 150px;
    height: 150px;
    background: #243E73;
    transition: all 0.4s ease;  /* 'all' anima todas as propriedades que mudarem */
}
.box:hover {
    transform: rotate(10deg) scale(1.1);
    background: #A67E6F;
}

Propriedades complementares: transition-delay (atraso), transition-timing-function (aceleração: ease, linear, ease-in-out).

💡 Dica profissional: Combine transformações com transições para criar efeitos sutis em cards, botões e menus. Exemplo: um card que sobe levemente (translateY(-5px)) e aumenta sombra no hover.

🚀 Micro-atividade 5: Card animado com transform e transition

Crie um card de produto (div com 200px de largura, borda arredondada, sombra). No hover, o card deve: (1) aumentar 5% (scale 1.05); (2) girar 2 graus (rotate(2deg)); (3) mudar a cor da borda suavemente em 0.3s. Utilize transição para todas as propriedades.

Resumo prático para consulta rápida

  • Seletores avançados: E > F (filho), E + F (irmão adjacente), :first-child.
  • Pseudoclasses comuns: :hover, :focus, :checked.
  • Pseudoelementos úteis: ::before, ::after, ::first-letter.
  • Menus: use <ul> com list-style: none, display: block nos links, e submenu com position: absolute.
  • Variáveis: :root { --nome: valor; } e var(--nome).
  • Transformações: translate(), rotate(), scale(), skew().
  • Transições: transition: propriedade duração timing; (ex: all 0.3s ease).
