Introdução ao JavaScript
Dê vida às suas páginas com lógica, interação e dinamismo

Com o JavaScript, suas páginas web deixam de ser estáticas e passam a interagir com o usuário: validar formulários, exibir mensagens, calcular valores, manipular o DOM e muito mais. Esta apostila oferece uma base sólida na linguagem: variáveis, tipos de dados, operadores, estruturas de decisão, laços de repetição e formas de interação com o usuário. Tudo com exemplos práticos que você pode testar no console do navegador ou em arquivos HTML.

1. JavaScript: o que é e como funciona

JavaScript é uma linguagem de programação interpretada, que roda diretamente no navegador do usuário (lado do cliente). Isso significa que, ao abrir uma página, o navegador executa o código JS para criar interatividade, sem precisar de comunicação com o servidor a cada ação.

📌 Ponto importante: JavaScript é diferente de Java. O nome foi uma estratégia de marketing na época. O padrão oficial da linguagem é chamado ECMAScript (versão atual ES2022+).

1.1 Ambiente de desenvolvimento e console

Para começar, você só precisa de um navegador e um editor de código (recomendamos VS Code). O navegador já possui um console JavaScript integrado, onde você pode testar trechos de código rapidamente.

  • Abrir console no Chrome/Edge/Firefox: tecla F12 ou Ctrl + Shift + J (Windows/Linux) / Cmd + Option + J (Mac).
  • No console, digite comandos e pressione Enter para executar.
// Exemplo: digite no console e veja o resultado
console.log("Olá, mundo!");
alert("Bem-vindo ao JavaScript!");

1.2 Onde colocar o JavaScript

Você pode inserir JS diretamente em um arquivo HTML usando a tag <script>, ou criar um arquivo separado com extensão .js e linká-lo.

<!-- Interno (dentro do HTML) -->
<script>
    console.log("JS interno");
</script>

<!-- Externo (recomendado para códigos maiores) -->
<script src="meu-script.js"></script>

🖥️ Micro-atividade 1: Primeiro contato com o console

Crie um arquivo HTML simples, abra o console do navegador (F12). Digite as seguintes linhas e observe os resultados: console.log("Meu nome é [seu nome]"), alert("Mensagem de boas-vindas"), 2 + 2 * 5. Depois, insira um bloco <script> no HTML e faça o mesmo.

2. Variáveis, tipos de dados e operadores

2.1 Declarando variáveis e constantes

Em JavaScript moderno, usamos let para variáveis que podem mudar e const para valores fixos (constantes). Evite usar var (método antigo).

"use strict"; // Força a declaração correta de variáveis
let nome = "Ana";
let idade = 25;
const PI = 3.14159;
let estaLogado = true;

2.3 Operadores essenciais

  • Aritméticos: + (soma/ concatenação), -, *, /, % (resto), ** (potência).
  • Atribuição: =, +=, -=, *=, etc.
  • Comparação: == (igual valor), === (igual valor e tipo), !=, !==, >, <, >=, <=.
  • Lógicos: && (E), || (OU), ! (NÃO).
let a = 10, b = 20;
console.log(a + b);        // 30
console.log(a === "10");   // false (tipos diferentes)
console.log(a == "10");    // true (conversão automática, evite)
console.log(a > 5 && b < 30); // true

2.4 Trabalhando com strings

Strings possuem métodos úteis: length, charAt(), slice(), split() e template strings (usando crase `` ` ``) para interpolar variáveis.

let curso = "JavaScript";
console.log(curso.length);        // 10
console.log(curso.charAt(0));     // "J"
console.log(curso.slice(0, 4));   // "Java"

// Template string (interpolação)
let aluno = "Maria";
console.log(`Bem-vinda, ${aluno}! Hoje é dia ${2 + 3}`);

3. Interação com o usuário (alert, confirm, prompt)

JavaScript oferece três funções básicas para interagir com o usuário. Use com moderação (são bloqueantes), mas são excelentes para aprendizado.

3.1 Alert – mensagem informativa

alert("Bem-vindo ao meu site!");

3.2 Confirm – pergunta com OK/Cancelar (retorna true/false)

let resposta = confirm("Você tem mais de 18 anos?");
if (resposta) {
    console.log("Acesso permitido");
} else {
    console.log("Acesso negado");
}

3.3 Prompt – entrada de texto do usuário

let nome = prompt("Qual é o seu nome?");
alert(`Olá, ${nome}! Seja bem-vindo(a).`);

💬 Micro-atividade 3: Saudação personalizada

Crie um arquivo HTML com um bloco <script>. Use prompt para perguntar o nome do usuário e a cidade onde mora. Depois, exiba uma mensagem personalizada com alert ou console.log no formato: "Olá, [nome]! Você mora em [cidade]."

4. Estruturas de controle de fluxo (decisões)

4.1 if / else / else if

Executa blocos de código conforme condições lógicas.

let idade = parseInt(prompt("Digite sua idade:"));
if (idade < 12) {
    console.log("Criança");
} else if (idade < 18) {
    console.log("Adolescente");
} else if (idade < 60) {
    console.log("Adulto");
} else {
    console.log("Idoso");
}

4.2 switch – múltiplas comparações com um valor

let dia = prompt("Digite um dia da semana (segunda, terça, ...)");
switch (dia) {
    case "segunda":
    case "terça":
    case "quarta":
    case "quinta":
    case "sexta":
        console.log("Dia útil");
        break;
    case "sábado":
    case "domingo":
        console.log("Fim de semana");
        break;
    default:
        console.log("Dia inválido");
}

🎯 Micro-atividade 4: Classificador de notas

Peça ao usuário que digite a nota de um aluno (0 a 100). Use estruturas if/else para classificar: A (>=90), B (>=80), C (>=70), D (>=60), F (abaixo de 60). Exiba o resultado com alert.

5. Laços de repetição (loops)

Loops executam um bloco de código várias vezes.

5.1 for – quando se sabe o número de repetições

// Exibe números de 0 a 9
for (let i = 0; i < 10; i++) {
    console.log(i);
}

// Tabuada do 5
let numero = 5;
for (let i = 1; i <= 10; i++) {
    console.log(`${numero} x ${i} = ${numero * i}`);
}

5.2 while – repete enquanto condição for verdadeira

let contador = 0;
while (contador < 5) {
    console.log(`Contador: ${contador}`);
    contador++;
}

5.3 do/while – executa ao menos uma vez

let senha;
do {
    senha = prompt("Digite a senha (1234):");
} while (senha !== "1234");
alert("Acesso liberado!");

🔄 Micro-atividade 5: Jogo de adivinhação

Gere um número aleatório com Math.floor(Math.random() * 10) + 1. Use um loop while que só para quando o usuário acertar o número. A cada tentativa, dê dicas ("Muito baixo", "Muito alto"). Ao final, exiba quantas tentativas foram necessárias.

6. Manipulação básica do DOM (acessando elementos HTML)

O DOM (Document Object Model) é a representação da página HTML em forma de árvore. Com JavaScript, podemos acessar elementos, ler valores e alterar conteúdos dinamicamente.

6.1 Selecionando elementos

<input type="text" id="nome" placeholder="Digite seu nome">
<button onclick="mostrarNome()">Enviar</button>
<p id="mensagem"></p>

<script>
function mostrarNome() {
    let campo = document.getElementById("nome");
    let nomeDigitado = campo.value;
    document.getElementById("mensagem").innerHTML = `Olá, ${nomeDigitado}!`;
}
</script>

6.2 Eventos comuns

  • onclick – quando o elemento é clicado.
  • onmouseover – mouse sobre o elemento.
  • onchange – quando um campo de formulário é alterado.
  • onload – quando a página termina de carregar.

📝 Micro-atividade 6: Formulário interativo simples

Crie uma página com: um campo de texto (para o nome), um botão "Exibir mensagem" e uma div vazia. Ao clicar no botão, o JavaScript deve ler o valor do campo e escrever dentro da div: "Bem-vindo, [nome]!". Adicione também um evento onmouseover no botão para mudar sua cor de fundo.

7. Dicas finais e boas práticas

  • Use "use strict" no início dos seus scripts para evitar erros silenciosos.
  • Nomeie variáveis de forma clara (ex: precoProduto, idadeUsuario).
  • Teste seus códigos no console antes de implementar no HTML.
  • Evite poluição do escopo global – mantenha variáveis dentro de funções ou blocos.
  • Comente seu código usando // para linha única ou /* ... */ para múltiplas linhas.
🧠 Desafio final (projeto prático): Crie uma calculadora simples de descontos. O usuário informa o valor do produto e a porcentagem de desconto via prompt. O programa deve calcular o valor final e exibir em uma janela alert ou diretamente no HTML. Use variáveis, operadores e estrutura condicional para validar se os valores são válidos.
