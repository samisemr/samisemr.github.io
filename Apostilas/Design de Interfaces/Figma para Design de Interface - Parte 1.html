🎨 Figma para Design de Interface · Parte 1

Primeiros passos e ferramentas essenciais

O Figma é a ferramenta de design de interface mais utilizada no mundo — colaborativa, baseada na nuvem e gratuita para estudantes. Nesta apostila prática, você aprenderá a navegar pela interface, criar seus primeiros frames, shapes, textos e exportar assets. Tudo com foco em projetos reais de UI Design.

1. Conhecendo o ambiente do Figma

Ao abrir o Figma (app desktop ou navegador), você verá o dashboard com seus arquivos. Crie um novo arquivo (File > New Design File) e vamos explorar as áreas principais.

🖥️
[IMAGEM: Print da interface do Figma destacando: Toolbar (topo), Layers (esquerda), Canvas (centro), Properties (direita)]
Figura 1 – Áreas do Figma: ferramentas, camadas, canvas e propriedades.

Elementos da interface

  • Toolbar (barra superior): ferramentas de seleção, formas, texto, caneta, mão, etc.
  • Layers (painel esquerdo): lista todos os objetos da página. Você pode organizar, renomear e agrupar.
  • Canvas (área central): seu espaço de trabalho infinito. Use Space + arrastar para navegar.
  • Properties (painel direito): ajusta largura, altura, cor, borda, sombra, texto, etc. Tudo que você selecionar aparece aqui.
Pressione Ctrl + \ (Windows) ou Cmd + \ (Mac) para esconder/mostrar a interface e focar no design. Use Shift + R para entrar no modo de apresentação (espelhar).
🧪 MICROATIVIDADE – Explorando o Figma Crie um novo arquivo. Identifique onde fica a ferramenta "Retângulo", o painel de camadas e o campo de opacidade no painel direito. Anote em um papel os nomes desses três elementos. Isso fixa a navegação!

2. Frames: a base do design de interface

Diferente de outras ferramentas, o Figma usa Frames (atalho F) como contêineres para telas. Ao contrário de formas comuns, frames permitem definir tamanhos fixos, aplicar grids automáticos e criar protótipos.

Criando seu primeiro frame

  1. Selecione a ferramenta Frame (F).
  2. No painel direito, escolha um presets (ex: iPhone 15 ou Desktop 1440×1024).
  3. Clique no canvas ou arraste para definir o tamanho.
  4. Nomeie o frame no painel de layers (duplo clique ou Enter).
📱
[IMAGEM: Canvas com dois frames: um mobile (390x844) e um desktop (1280x832), ambos nomeados]
Figura 2 – Frames representando telas de diferentes dispositivos.
Frames também podem conter outros frames (aninhamento). Isso é útil para criar componentes como cards, headers ou barras laterais dentro de uma tela principal.

Diferença entre Frame e Grupo

Frame (F)Grupo (Ctrl+G)
Possui tamanho definido e clipping (corta objetos que ultrapassam)Apenas organiza objetos, sem cortar ou limitar área
Pode ter grids, layout grids e auto layoutNão suporta grids ou auto layout nativamente
Ideal para telas, botões, cards, componentes reutilizáveisIdeal para agrupar temporariamente elementos
Exporta como imagem respeitando o tamanho do frameExporta a área total dos objetos, sem moldura
Prefira frames em vez de grupos quando estiver projetando interfaces. Isso facilita a prototipagem e o uso de auto layout (veremos na Parte 2).

3. Shapes, textos e propriedades visuais

Com os frames criados, é hora de adicionar elementos. As principais ferramentas de desenho estão na toolbar ou via atalhos.

Ferramentas básicas

  • Retângulo (R) – formas quadradas ou retangulares.
  • Linha (L) – traços retos.
  • Elipse (O) – círculos e ovais.
  • Texto (T) – clique e digite ou arraste para criar uma caixa de texto.
🔘
[IMAGEM: Exibindo um retângulo, uma elipse e um texto editável com fonte Inter, 16px]
Figura 3 – Elementos primitivos e texto no canvas.

Ajustando propriedades (painel direito)

Selecione qualquer objeto e no painel direito você encontrará:

  • Posição (X, Y) e Tamanho (W, H) – valores numéricos precisos.
  • Preenchimento (Fill) – cor sólida, gradiente ou imagem.
  • Stroke (Borda) – espessura, cor, tipo (tracejado, pontilhado).
  • Efeitos – sombras, desfoque.
  • Camada de texto – família, peso, tamanho, altura da linha, espaçamento entre caracteres.
Use o seletor de cores para aplicar cores hexadecimais (#FFFFFF, #A259FF) ou cores do sistema. Você também pode criar estilos de cores e textos (veremos na Parte 2).
🎨 EXERCÍCIO – Criando um botão simples Dentro de um frame mobile (390×844), desenhe um retângulo de 200×50px. Aplique cor de fundo azul (#2C7DA0), borda arredondada de 12px e adicione um texto centralizado “Clique aqui” na cor branca. Salve o arquivo (ele salva automaticamente na nuvem).

4. Organização com Layers e seleção

Projetos reais possuem dezenas ou centenas de camadas. Organizar desde o início evita dores de cabeça.

Boas práticas no painel Layers

  • Renomeie camadas: duplo clique no nome. Use nomes descritivos: “btn_primario”, “header_bg”, “icone_usuario”.
  • Agrupe com Frame: selecione elementos e pressione Ctrl+Alt+G (cria um frame ao redor).
  • Bloqueie camadas: clique no cadeado ao lado da layer para evitar edição acidental.
  • Oculte camadas: clique no ícone de olho para esconder temporariamente.

5. Exportando assets para desenvolvimento

Após criar ícones, botões ou imagens, você precisará exportá-los como PNG, JPG, SVG ou PDF. O Figma facilita esse processo.

Passo a passo para exportar

  1. Selecione o frame, grupo ou shape desejado.
  2. No painel direito, role até a seção Export.
  3. Clique no botão “+” para adicionar um formato.
  4. Escolha o formato (PNG é o mais comum para interfaces).
  5. Ajuste o tamanho (1x, 2x, 3x para telas retina).
  6. Clique em Exportar e salve no seu computador.
FormatoQuando usar
PNGImagens com fundo transparente, ícones, botões (qualidade com perda leve).
JPGFotos, backgrounds (não suporta transparência).
SVGÍcones vetoriais (infinitamente redimensionáveis). Ótimo para devs.
PDFApresentações, entregáveis para clientes.
Para exportar múltiplos assets de uma vez, selecione todos e configure a exportação. O Figma exportará cada um em um arquivo separado automaticamente.
📤 MICROATIVIDADE – Exporte seu botão Exporte o botão azul que você criou no exercício anterior em dois formatos: PNG (1x) e SVG. Compare os arquivos resultantes. Qual pesa menos? Qual mantém a nitidez ao ampliar?

📌 Resumo – Parte 1

  • O Figma tem 4 áreas principais: Toolbar, Layers, Canvas, Properties.
  • Frames são contêineres de tela (prefira frames a grupos).
  • Use as ferramentas R, L, O, T para criar formas e textos.
  • O painel direito permite ajustar cores, bordas, sombras e tipografia.
  • Organize camadas com nomes descritivos e hierarquia.
  • Exporte assets em PNG, SVG ou JPG conforme a necessidade.
