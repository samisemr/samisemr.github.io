🧩 Figma para Design de Interface · Parte 2

Componentes, Auto Layout e Design System

Na Parte 1, você aprendeu a navegar no Figma, criar frames, formas, textos e exportar assets. Agora vamos elevar seu nível: componentes reutilizáveis, auto layout (responsividade mágica) e os primeiros passos para construir um Design System. Esses recursos separam o design amador do profissional.

1. Componentes: crie uma vez, use muitas vezes

Componentes são elementos mestres que podem ser reutilizados em todo o projeto. Quando você edita o componente principal, todas as suas instâncias são atualizadas automaticamente. Isso é fundamental para manter consistência em botões, ícones, cards e headers.

🧩
[IMAGEM: Componente principal (diamante roxo) e três instâncias (diamante vazado) em diferentes telas]
Figura 1 – Componente principal (ícone de diamante) e instâncias espalhadas pelo projeto.

Criando e usando componentes

  1. Desenhe um elemento (ex: botão com texto “Enviar”).
  2. Selecione o elemento e clique no ícone de diamante na toolbar (ou pressione Ctrl+Alt+K / Cmd+Option+K).
  3. O frame fica com borda roxa (quatro diamantes no contorno) – agora é um Component Master.
  4. Para criar instâncias, copie (Ctrl+C / Ctrl+V) ou arraste segurando Alt.
  5. Edite o componente principal: todas as instâncias se atualizam.
Você pode sobrescrever propriedades em uma instância específica (ex: mudar o texto de um botão de “Enviar” para “Cancelar”). Isso não afeta o componente principal. Para resetar, clique no ícone de “reset” no painel de propriedades.

Variants (variantes): organize componentes similares

Variantes permitem agrupar diferentes estados ou tamanhos de um mesmo componente. Exemplo: um botão com variantes Primário / Secundário / Desabilitado e Pequeno / Médio / Grande.

Exemplo de estrutura de variantes:
Botão / Estado [Primário | Secundário | Desabilitado] / Tamanho [P | M | G]

2. Auto Layout: design responsivo sem sofrimento

Auto layout é o recurso mais amado do Figma. Ele transforma frames em contêineres inteligentes que se redimensionam automaticamente conforme o conteúdo ou o espaço disponível. É como ter CSS Flexbox dentro do Figma.

Ativando Auto Layout

  • Selecione um frame ou grupo de elementos.
  • Pressione Shift+A (atalho universal).
  • Ou clique com botão direito → Add Auto Layout.
📐
[IMAGEM: Dois cards lado a lado – sem auto layout (fixos) vs com auto layout (se ajustam ao texto e à largura)]
Figura 3 – Auto layout permite que componentes se reorganizem dinamicamente.

Propriedades essenciais do Auto Layout

PropriedadeO que controlaExemplo
DirectionHorizontal ou verticalBotões em linha (horizontal) vs. formulário empilhado (vertical)
SpacingDistância entre elementos internos8px entre cada campo de texto
PaddingEspaçamento interno entre borda do frame e elementos16px em todas as laterais de um card
Resizing (Fill / Fixed / Hug)Como cada elemento se comportaBotão “Hug” (abraça conteúdo) / Campo “Fill” (ocupa largura disponível)
O Auto Layout também suporta Wrap (quebra de linha) – útil para galerias de imagens ou tags que precisam se reorganizar em múltiplas linhas quando a tela encolhe.
📐 EXERCÍCIO – Criando um formulário responsivo Crie um frame vertical com Auto Layout (Shift+A). Adicione: label “Nome”, campo de texto (retângulo com Fill horizontal), label “E-mail”, campo de texto, e um botão “Enviar” com Hug. Teste redimensionar o frame – veja os campos se expandirem e o botão permanecer no tamanho do texto.

3. Layout Grids: estrutura profissional para telas

Assim como vimos na parte teórica, grids organizam o conteúdo. No Figma, você pode adicionar grids a qualquer frame.

Adicionando grid a um frame

  1. Selecione um frame (ex: tela desktop 1280×832).
  2. No painel direito, na seção Layout grid, clique no ícone “+” (mais).
  3. Escolha o tipo: Grid (quadriculado), Columns (colunas) ou Rows (linhas).
  4. Para design responsivo, use Columns com 12 colunas, largura variável e gutters (espaços) de 16px.
📏
[IMAGEM: Frame desktop com grid de 12 colunas visível; elementos de interface alinhados às colunas]
Figura 4 – Grid de 12 colunas guiando o posicionamento de cards e texto.
Você pode salvar grids como estilos (ícone de 4 pontos ao lado de Layout grid) para reutilizar em várias telas. Isso garante consistência entre todas as páginas do projeto.

4. Construindo um Design System básico

Um Design System é um conjunto de padrões reutilizáveis: cores, tipografia, sombras, espaçamentos e componentes. O Figma permite criar estilos e componentes que formam a base do seu sistema.

Estilos de cor

Selecione um elemento com a cor desejada → no painel de Fill, clique no ícone de 4 pontos → “+” → nomeie (ex: “Primary / Azul Principal”). Agora qualquer frame pode aplicar esse estilo.

Estilos de texto

Selecione um texto → no painel de texto, clique no ícone de 4 pontos → “+” → crie estilos como “Heading 1”, “Body Large”, “Caption”.

Estilos de efeito (sombra, desfoque)

Em Effects, clique no ícone de 4 pontos para salvar uma sombra padrão.

5. Plugins que vão acelerar seu trabalho

O Figma possui uma comunidade enorme de plugins (Menu → Plugins → Browse Plugins). Alguns indispensáveis para iniciantes:

PluginFunção
UnsplashInsere imagens stock de alta qualidade diretamente no seu design.
IconifyBiblioteca com milhares de ícones (Material, Feather, Font Awesome).
AutoflowCria setas de fluxo entre telas – ótimo para wireframes.
StarkVerifica contraste de cores e simula daltonismo.
Content ReelPreenche textos placeholder (lorem ipsum, nomes, e-mails) automaticamente.
Para instalar um plugin: vá em Plugins → Browse plugins, pesquise, clique em “Install”. Depois, aparecerá no menu Plugins para uso imediato.

📌 Resumo – Parte 2

  • Componentes (Ctrl+Alt+K) centralizam a manutenção de UI. Use variantes para organizar estados.
  • Auto Layout (Shift+A) torna o design responsivo e eficiente (Fill, Hug, Fixed).
  • Layout Grids (colunas, gutters) garantiam alinhamento profissional.
  • Estilos de cor, texto e efeito formam a base do Design System.
  • Plugins (Unsplash, Iconify, Stark) aumentam produtividade.
