Estruturas de Dados em Python
Tuplas, Listas, Dicionários e Métodos Avançados de Strings

Até agora, trabalhávamos com variáveis simples que armazenam um único valor. Mas e quando precisamos armazenar múltiplos valores relacionados? Para isso, usamos estruturas de dados. Nesta apostila, você aprenderá as três principais estruturas do Python: tuplas (imutáveis), listas (mutáveis e versáteis) e dicionários(pares chave-valor). Também exploraremos métodos avançados de strings para manipulação de texto.

1. Tuplas – estruturas imutáveis

Uma tupla é uma coleção ordenada e imutável de elementos. Uma vez criada, não pode ser alterada (não é possível adicionar, remover ou modificar elementos). É ideal para dados que não devem mudar, como dias da semana, coordenadas geográficas, etc.

1.1 Criando e acessando tuplas

# Tupla com parênteses (opcional, mas recomendado)
frutas = ("maçã", "banana", "laranja")
print(frutas)          # ('maçã', 'banana', 'laranja')

# Tupla sem parênteses (também funciona)
coordenadas = 10, 20, 30
print(coordenadas)     # (10, 20, 30)

# Acesso por índice (começa em 0)
print(frutas[0])       # maçã
print(frutas[-1])      # laranja (último elemento)

# Fatiamento (slicing)
print(frutas[0:2])     # ('maçã', 'banana')

1.2 Imutabilidade

# Tentativa de modificar (gera erro)
frutas = ("maçã", "banana", "laranja")
# frutas[0] = "uva"  # TypeError: 'tuple' object does not support item assignment

# Para "alterar", é necessário criar uma nova tupla
novas_frutas = frutas + ("uva",)
print(novas_frutas)    # ('maçã', 'banana', 'laranja', 'uva')

1.3 Iterando sobre tuplas

# Usando for
for fruta in frutas:
    print(fruta)

# Usando for com índice
for i in range(len(frutas)):
    print(f"{i}: {frutas[i]}")

1.4 Desempacotamento de tuplas

coordenadas = (10, 20, 30)
x, y, z = coordenadas
print(f"x={x}, y={y}, z={z}")  # x=10, y=20, z=30

# Desempacotamento com * (resto)
numeros = (1, 2, 3, 4, 5)
primeiro, *resto = numeros
print(primeiro)  # 1
print(resto)     # [2, 3, 4, 5] (lista)

2. Listas – estruturas mutáveis e versáteis

Listas são coleções ordenadas e mutáveis. Podemos adicionar, remover e modificar elementos. São as estruturas mais usadas em Python para sequências dinâmicas.

2.1 Criando e acessando listas

# Lista com colchetes
frutas = ["maçã", "banana", "laranja"]
print(frutas)          # ['maçã', 'banana', 'laranja']

# Lista vazia
vazia = []
vazia2 = list()

# Acesso por índice (igual tuplas)
print(frutas[0])       # maçã
frutas[0] = "uva"      # modificação permitida
print(frutas)          # ['uva', 'banana', 'laranja']

2.2 Métodos principais de listas

MétodoDescriçãoExemplo
append()么Adiciona no final么frutas.append("pera")
insert()么Insere em posição específica么frutas.insert(1, "manga")
remove()么Remove pelo valor (primeira ocorrência)么frutas.remove("banana")
pop()么Remove e retorna o último (ou índice)么ultimo = frutas.pop()
index()么Retorna índice do valor么frutas.index("laranja")
sort()么Ordena a lista么numeros.sort()
reverse()么Inverte a ordem么frutas.reverse()
len()么Tamanho da lista (função)么len(frutas)

2.3 Exemplos práticos com listas

# Lista de números
numeros = [3, 1, 4, 1, 5, 9, 2]

# Adicionar elementos
numeros.append(6)           # [3, 1, 4, 1, 5, 9, 2, 6]
numeros.insert(2, 99)       # [3, 1, 99, 4, 1, 5, 9, 2, 6]

# Remover elementos
numeros.remove(1)           # remove o primeiro 1
ultimo = numeros.pop()      # remove e retorna 6

# Ordenar
numeros.sort()              # [1, 2, 3, 4, 5, 9, 99]
numeros.reverse()           # [99, 9, 5, 4, 3, 2, 1]

# Verificar existência
if 5 in numeros:
    print("5 está na lista")

2.4 Cópia de listas (cuidado!)

# ATENÇÃO: isso cria uma referência, não uma cópia
lista1 = [1, 2, 3]
lista2 = lista1          # lista2 aponta para o mesmo objeto
lista2[0] = 99
print(lista1)            # [99, 2, 3] (lista1 também foi alterada!)

# Para copiar de verdade:
lista2 = lista1.copy()   # ou lista1[:]
lista2[0] = 100
print(lista1)            # [99, 2, 3] (não altera)

2.5 Listas aninhadas (listas de listas)

# Matriz 3x3
matriz = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]
print(matriz[0][1])      # 2 (linha 0, coluna 1)

# Lista de compras com nome, quantidade, preço
compras = []
compras.append(["Arroz", 2, 25.90])
compras.append(["Feijão", 3, 8.50])
for item in compras:
    print(f"{item[0]}: {item[1]} x R$ {item[2]} = R$ {item[1] * item[2]:.2f}")

3. Dicionários – pares chave-valor

Dicionários armazenam dados em pares chave: valor. Em vez de índices numéricos, usamos chaves (geralmente strings) para acessar os valores. São ideais para representar registros (como um cadastro de aluno).

3.1 Criando e acessando dicionários

# Dicionário com chaves e valores
aluno = {
    "nome": "Ana Silva",
    "idade": 20,
    "curso": "Engenharia"
}
print(aluno)               # {'nome': 'Ana Silva', 'idade': 20, 'curso': 'Engenharia'}

# Acesso por chave
print(aluno["nome"])       # Ana Silva
print(aluno["idade"])      # 20

# Adicionar/modificar
aluno["nota"] = 8.5        # adiciona nova chave
aluno["idade"] = 21        # modifica valor existente

# Dicionário vazio
vazio = {}
vazio2 = dict()

3.2 Métodos de dicionários

# keys() - retorna todas as chaves
print(aluno.keys())        # dict_keys(['nome', 'idade', 'curso', 'nota'])

# values() - retorna todos os valores
print(aluno.values())      # dict_values(['Ana Silva', 21, 'Engenharia', 8.5])

# items() - retorna pares (chave, valor)
for chave, valor in aluno.items():
    print(f"{chave}: {valor}")

# get() - acesso seguro (não gera erro se chave não existir)
print(aluno.get("endereco", "Não informado"))  # Não informado

# Verificar se chave existe
if "nome" in aluno:
    print("Nome está cadastrado")

# Remover
del aluno["nota"]          # remove a chave "nota"
idade = aluno.pop("idade") # remove e retorna o valor

3.3 Listas de dicionários (cadastro de múltiplos itens)

# Lista de alunos (cada aluno é um dicionário)
alunos = []

while True:
    nome = input("Nome do aluno (ou 'sair'): ")
    if nome.lower() == 'sair':
        break
    idade = int(input("Idade: "))
    nota = float(input("Nota: "))
    
    aluno = {"nome": nome, "idade": idade, "nota": nota}
    alunos.append(aluno)

# Exibir todos os alunos
print("\n--- Lista de Alunos ---")
for a in alunos:
    print(f"{a['nome']} - {a['idade']} anos - Nota: {a['nota']}")

4. Métodos avançados de strings

Strings também possuem métodos úteis para manipulação e validação de texto.

4.1 Verificação de início/fim e conversão de caso

texto = "Python é uma linguagem poderosa"

# startswith / endswith
print(texto.startswith("Python"))   # True
print(texto.endswith("poderosa"))   # True

# upper / lower
print(texto.upper())   # PYTHON É UMA LINGUAGEM PODEROSA
print(texto.lower())   # python é uma linguagem poderosa
print(texto.capitalize())  # Python é uma linguagem poderosa

# count - conta ocorrências
print(texto.count("a"))          # 4
print(texto.lower().count("a"))  # 5 (inclui maiúscula)

4.2 Validação de conteúdo

# isalpha() - só letras (sem espaços)
print("Python".isalpha())      # True
print("Python3".isalpha())     # False (tem número)
print("Python é".isalpha())    # False (tem espaço)

# isdigit() - só dígitos
print("123".isdigit())         # True
print("12.3".isdigit())        # False (ponto)

# isalnum() - letras e números (sem espaços)
print("Python3".isalnum())     # True
print("Python 3".isalnum())    # False (espaço)

# isspace() - só espaços
print("   ".isspace())         # True

4.3 Split, join e replace

# split - divide string em lista
frase = "maçã,banana,laranja,uva"
lista = frase.split(",")
print(lista)        # ['maçã', 'banana', 'laranja', 'uva']

# join - junta lista em string
nova_frase = "-".join(lista)
print(nova_frase)   # maçã-banana-laranja-uva

# replace - substitui
texto = "O rato roeu a roupa do rei de Roma"
novo = texto.replace("rato", "gato")
print(novo)         # O gato roeu a roupa do rei de Roma

# strip - remove espaços das extremidades
texto_com_espacos = "   Olá   "
print(texto_com_espacos.strip())  # "Olá"

4.4 Find e centralização

# find - encontra posição da primeira ocorrência
texto = "Python é incrível, Python é poderoso"
pos = texto.find("Python")
print(pos)          # 0 (primeira ocorrência)
print(texto.find("Java"))   # -1 (não encontrado)

# center - centraliza texto
titulo = "MENU"
print(titulo.center(20, "-"))   # --------MENU---------

5. Comparativo entre as estruturas

CaracterísticaTuplaListaDicionário
Mutabilidade么Imutável么Mutável么Mutável
Índice么Numérico (0, 1, 2...)么Numérico (0, 1, 2...)么Chave (string, int, etc.)
Sintaxe么()么[]么{}
Velocidade de busca么Mais rápida么Razoável么Muito rápida (hash)
Uso típico么Dados constantes, coordenadas么Sequências dinâmicas么Registros, catálogos
🧠 Resumo prático:
  • Use tuplas para dados que não devem mudar (dias da semana, coordenadas, constantes).
  • Use listas para sequências que precisam ser alteradas (lista de tarefas, notas de alunos).
  • Use dicionários para dados que serão acessados por chave (cadastro de clientes, configurações).

Exercício 1: Busca sequencial em lista

def busca_sequencial(lista, valor):
    for i in range(len(lista)):
        if lista[i] == valor:
            return i
    return -1

numeros = [10, 20, 30, 40, 50]
print(busca_sequencial(numeros, 30))  # 2
print(busca_sequencial(numeros, 99))  # -1
