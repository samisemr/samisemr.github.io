1.2 O primeiro programa: "Olá, Mundo!"

A tradição na programação é começar exibindo essa mensagem. Em Python, usamos a função print().

print('Olá, Mundo!')
Saída: Olá, Mundo!

1.3 Regras básicas do Python

  • Case sensitive: print é diferente de Print. Use sempre minúsculas.
  • Parênteses e aspas: toda abertura deve ter fechamento correspondente.
  • Indentação: o Python usa espaços para organizar blocos de código (veremos mais adiante).

2. Ciclo de processamento e função de saída (print)

Todo programa computacional segue três etapas: entrada (receber dados), processamento (manipular/calcular) e saída (exibir resultados). Vamos começar pela saída.

2.1 A função print

print() exibe mensagens na tela. Pode receber textos (entre aspas), números, variáveis e até resultados de operações.

# Texto literal
print('Olá, mundo!')

# Números e operações
print(2 + 3)        # Saída: 5
print(10 * 2)       # Saída: 20

# Múltiplos itens (separados por vírgula)
print('O resultado é', 2 + 3)   # Saída: O resultado é 5

# Concatenação com + (junta strings)
print('2' + '3')    # Saída: 23 (concatenação, não soma)

3. Variáveis e tipos de dados

Variáveis são espaços na memória que armazenam valores. Em Python, você não precisa declarar o tipo – ele é inferido automaticamente.

3.1 Regras para nomes de variáveis

  • Podem conter letras, números e sublinhado (_).
  • Não podem começar com número.
  • Não podem conter espaços ou caracteres especiais (@, #, $).
  • Case sensitive: nome é diferente de Nome.
  • Exemplos válidos: idade, nota_final, _valor, precoProduto.

3.2 Tipos primitivos

4. Manipulação de strings (textos)

Strings são sequências de caracteres. Python oferece várias facilidades para manipulá-las.

4.1 Concatenação e repetição

# Concatenação com +
nome = "Ana"
sobrenome = "Silva"
nome_completo = nome + " " + sobrenome
print(nome_completo)  # Ana Silva

# Repetição com *
linha = "-" * 20
print(linha)          # --------------------

4.2 Acessando caracteres por índice

O primeiro caractere está no índice 0 (zero).

texto = "Python"
print(texto[0])   # P
print(texto[1])   # y
print(texto[-1])  # n (último caractere)

4.3 Fatiamento (slicing)

texto = "Lógica de Programação"
print(texto[0:6])   # Lógica (índices 0 a 5)
print(texto[:6])    # Lógica (do início até o 5)
print(texto[10:])   # Programação (do 10 até o fim)
print(texto[:])     # string completa

4.4 Tamanho da string (len)

frase = "Olá, mundo!"
tamanho = len(frase)
print(tamanho)      # 12 (inclui espaços e pontuação)

4.5 Três formas de formatar strings

Maneira clássica (estilo C): usando % e marcadores (%s para string, %d para inteiro, %f para float).

nome = "Carlos"
idade = 25
print("Meu nome é %s e tenho %d anos." % (nome, idade))

Maneira moderna (format): mais flexível, usando {} e .format().

print("Meu nome é {} e tenho {} anos.".format(nome, idade))

Maneira recomendada (f-string): disponível a partir do Python 3.6, mais legível.

print(f"Meu nome é {nome} e tenho {idade} anos.")

5. Função de entrada (input) e conversão de tipos

A função input() permite receber dados digitados pelo usuário. Por padrão, ela sempre retorna uma string (texto). Para usar números, precisamos converter.

5.1 Usando input

nome = input("Digite seu nome: ")
print(f"Olá, {nome}! Seja bem-vindo(a).")

5.2 Conversão de tipos (casting)

# Convertendo para inteiro
idade = int(input("Digite sua idade: "))

# Convertendo para float
altura = float(input("Digite sua altura: "))

# Convertendo para string (geralmente não necessário)
texto = str(123)   # "123"

5.3 Exemplo completo: soma de dois números

# Entrada
num1 = int(input("Digite o primeiro número: "))
num2 = int(input("Digite o segundo número: "))

# Processamento
soma = num1 + num2

# Saída
print(f"A soma de {num1} e {num2} é {soma}.")
⚠️ Atenção: Se você esquecer de converter e tentar somar strings, o Python irá concatenar (juntar) ao invés de somar. Exemplo: "2" + "3" resulta em "23".

6. Fluxo de execução e teste de mesa

O Python executa as instruções em ordem, de cima para baixo, uma linha após a outra. Entender isso é fundamental para prever o comportamento do programa.

# Exemplo de fluxo
x = 10            # Linha 1: x recebe 10
y = 5             # Linha 2: y recebe 5
z = x + y         # Linha 3: z recebe 15
print(z)          # Linha 4: exibe 15

Teste de mesa é a técnica de simular manualmente a execução do programa, anotando os valores das variáveis linha a linha. É muito útil para encontrar erros de lógica.
