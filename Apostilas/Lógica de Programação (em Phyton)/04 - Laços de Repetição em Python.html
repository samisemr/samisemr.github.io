Laços de Repetição em Python
while, for, contadores, acumuladores e laços aninhados

Muitas tarefas na programação exigem que um bloco de código seja executado várias vezes. Imagine escrever 1000 vezes o mesmo comando "Mover" para um personagem andar 1000 passos – seria impraticável. As estruturas de repetição (laços ou loops) resolvem esse problema. Nesta apostila, você aprenderá while (repetição com condição), for (repetição com contador), contadores e acumuladores, além de laços aninhados e comandos como break e continue.

1. Por que usar estruturas de repetição?

Suponha que você precise imprimir os números de 1 a 1000 na tela. Fazer isso com 1000 prints seria um trabalho enorme e geraria um código gigantesco. Com um laço de repetição, escrevemos apenas algumas linhas e o computador se encarrega de repetir.

🧭 Exemplo lúdico: Um lenhador precisa andar 5 quadrados até a tora de madeira. Em vez de escrever "Mover" cinco vezes, usamos um laço que repete a ação 5 vezes. Se fossem 1000 passos, a mesma estrutura funciona – basta mudar o limite.

2. Estrutura de repetição while (enquanto)

O comando while repete um bloco de código enquanto uma condição for verdadeira. A repetição termina quando a condição se torna falsa.

2.1 Sintaxe e fluxograma

while condicao:
    # bloco executado repetidamente
    instrucao_1
    instrucao_2
🔷 [FLUXOGRAMA DO WHILE]
→ Condição? ── Sim → Bloco de instruções → (volta para condição)
          ↓ Não
→ Fim do laço (continua o programa)

2.2 Exemplo: imprimir números de 1 a 5

contador = 1
while contador <= 5:
    print(contador)
    contador = contador + 1   # incremento (importante para não gerar loop infinito)
Saída:
1
2
3
4
5
⚠️ Cuidado com loops infinitos! Se esquecer de atualizar a variável de controle (como o contador += 1), a condição nunca se torna falsa e o programa executa para sempre.

2.3 Contadores (incremento constante)

Um contador é uma variável que aumenta (ou diminui) de um valor fixo a cada repetição. É usado para controlar o número de iterações.

# Contagem regressiva de 10 a 1
contador = 10
while contador >= 1:
    print(contador)
    contador -= 1   # equivalente a contador = contador - 1
print("Fogo!")

2.4 Acumuladores (somatório de valores variáveis)

Um acumulador é uma variável que soma valores diferentes a cada repetição. É muito usado para calcular somatórios, médias, etc.

# Calcula a soma de 5 números digitados pelo usuário
soma = 0
contador = 1
while contador <= 5:
    numero = float(input(f"Digite o {contador}º número: "))
    soma = soma + numero
    contador += 1
print(f"Somatório: {soma}")
print(f"Média: {soma / 5}")

3. Validação de dados com while

Uma aplicação muito comum do while é validar entradas do usuário, repetindo a pergunta até que um valor válido seja fornecido.

3.1 Exemplo: garantindo um número positivo

numero = int(input("Digite um número positivo: "))
while numero <= 0:
    print("Valor inválido! O número deve ser positivo.")
    numero = int(input("Digite um número positivo: "))
print(f"Valor aceito: {numero}")

3.2 Exemplo: repetindo até digitar "sair"

print("Digite uma mensagem (ou 'sair' para encerrar):")
texto = input()
while texto != "sair":
    print(f"Você digitou: {texto}")
    texto = input()

4. Comandos break e continue

break e continue alteram o fluxo normal de um laço de repetição.

4.1 break – interrompe o laço imediatamente

# Encerra quando o usuário digitar "sair"
while True:   # loop infinito (intencional)
    texto = input("Digite algo (ou 'sair'): ")
    if texto == "sair":
        break   # sai do while
    print(f"Você digitou: {texto}")
print("Programa encerrado.")

4.2 continue – pula para a próxima iteração

# Só exibe números ímpares (pula os pares)
numero = 0
while numero < 10:
    numero += 1
    if numero % 2 == 0:
        continue   # volta ao início do while sem executar o print
    print(numero)
Saída: 1, 3, 5, 7, 9

5. Estrutura de repetição for (para)

O for é usado quando sabemos quantas vezes o laço deve repetir. Ele é ideal para percorrer sequências (como intervalos de números) e é mais compacto que o while nesses casos.

5.1 Sintaxe básica com range()

for variavel in range(inicio, fim, passo):
    # bloco executado para cada valor
  • range(5) → 0, 1, 2, 3, 4 (inicia em 0, termina antes do 5)
  • range(1, 6) → 1, 2, 3, 4, 5
  • range(2, 11, 2) → 2, 4, 6, 8, 10 (passo 2)
  • range(10, 0, -1) → 10, 9, 8, ..., 1 (passo negativo)

5.2 Exemplos práticos

# Exemplo 1: imprimir de 1 a 10
for i in range(1, 11):
    print(i)

# Exemplo 2: tabuada do 5
numero = 5
for i in range(1, 11):
    print(f"{numero} x {i} = {numero * i}")

# Exemplo 3: contagem regressiva
for i in range(10, 0, -1):
    print(i)
print("Fogo!")

5.3 Comparativo while vs for

Ambos resolvem o mesmo problema, mas for é mais conciso quando o número de repetições é conhecido.

# Usando while
i = 1
while i <= 5:
    print(i)
    i += 1

# Usando for (mais simples)
for i in range(1, 6):
    print(i)

6. Laços aninhados

Podemos colocar um laço dentro de outro. O laço interno é completamente executado para cada iteração do laço externo.

6.1 Exemplo: tabuada completa de 1 a 10

# Laço externo: números de 1 a 10
for numero in range(1, 11):
    print(f"\n--- Tabuada do {numero} ---")
    # Laço interno: multiplicadores de 1 a 10
    for i in range(1, 11):
        print(f"{numero} x {i} = {numero * i}")

O laço externo executa 10 vezes. Para cada uma dessas 10 execuções, o laço interno executa 10 vezes. Total: 100 iterações do print.

6.2 Exemplo: números primos

# Encontra números primos entre 2 e 50
for numero in range(2, 51):
    primo = True
    for divisor in range(2, numero):
        if numero % divisor == 0:
            primo = False
            break
    if primo:
        print(f"{numero} é primo")

7. Truthy e Falsey

Em Python, valores não booleanos podem ser tratados como True ou False em condições:

  • Falsey (considerados falsos): 0, 0.0, "" (string vazia), None, [] (lista vazia), {} (dicionário vazio).
  • Truthy (considerados verdadeiros): qualquer outro valor (números diferentes de zero, strings não vazias, etc.)

7.1 Exemplo prático

nome = ""
while not nome:   # enquanto nome estiver vazio (Falsey)
    nome = input("Digite seu nome (não pode ficar em branco): ")
print(f"Olá, {nome}!")

valor = int(input("Digite um número: "))
if valor:   # equivalente a "if valor != 0"
    print("Você digitou um número diferente de zero.")
else:
    print("Você digitou zero.")

Exercício 4: Validando intervalo de horas

h_inicio = int(input("Hora inicial (0-23): "))
h_fim = int(input("Hora final (0-23): "))

while h_inicio < 0 or h_inicio > 23 or h_fim < 0 or h_fim > 23 or h_inicio >= h_fim:
    print("Intervalo inválido! A hora inicial deve ser menor que a final e entre 0 e 23.")
    h_inicio = int(input("Hora inicial: "))
    h_fim = int(input("Hora final: "))

for hora in range(h_inicio, h_fim + 1):

Resumo da apostila

  • while: repete enquanto condição for verdadeira. Ideal para número indeterminado de iterações ou validação de dados.
  • for: repete para cada elemento de uma sequência (range, string, lista). Ideal para número conhecido de iterações.
  • Contador: variável que incrementa valor constante (ex: i += 1).
  • Acumulador: variável que soma valores variáveis (ex: soma += valor).
  • break: interrompe o laço imediatamente.
  • continue: pula para a próxima iteração.
  • Truthy/Falsey: 0, "", None, etc. são falsos; outros valores são verdadeiros.
  • Operadores especiais: +=, -=, *=, /=, %=.
