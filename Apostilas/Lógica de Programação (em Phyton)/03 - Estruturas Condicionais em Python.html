PYTHON | ESTRUTURAS CONDICIONAIS

Estruturas Condicionais em Python
if, else, elif, expressões lógicas e tomada de decisões

Até agora, nossos programas executavam todas as instruções em sequência, sem qualquer variação. Mas a maioria dos problemas do mundo real exige decisões: "se estiver chovendo, levo guarda-chuva; senão, não levo". Nesta apostila, você aprenderá as estruturas condicionais em Python: if (simples), if/else (composta), elif (múltipla escolha), além de expressões lógicas com operadores and, or, not e condicionais aninhadas. Com esses recursos, seus programas poderão tomar decisões inteligentes e se adaptar a diferentes situações.

1. Algoritmos sequenciais vs. algoritmos com decisão

Um algoritmo sequencial executa todas as instruções na ordem em que foram escritas, sem desvios. Já um algoritmo com estruturas de decisão pode executar diferentes blocos de código dependendo de uma condição.

🧭 Exemplo lúdico: Um lenhador vai até a floresta cortar lenha. Se ele escolher o caminho da esquerda, encontra um lobo e precisa enfrentá-lo. Se escolher o caminho da direita, não encontra obstáculos. A decisão (esquerda/direita) altera o fluxo do algoritmo.

1.1 Fluxograma de uma decisão

🔷 [FLUXOGRAMA - ESTRUTURA DE DECISÃO]
Início → Entrada de dados → Losango (condição?) → Sim → Ação A → Fim
                             ↓ Não
                             Ação B → Fim

2. Condicional simples (if)

A estrutura condicional simples executa um bloco de código apenas se a condição for verdadeira. Se for falsa, o bloco é ignorado e o programa continua.

2.1 Sintaxe em Python

if condicao:
    # bloco executado se condicao for True
    instrucao_1
    instrucao_2

Importante: O bloco deve estar indentado (recuado) – em Python, a indentação define o escopo do bloco!

2.2 Exemplo prático

# Verifica se o primeiro número é maior que o segundo
x = int(input("Digite o primeiro valor: "))
y = int(input("Digite o segundo valor: "))

if x > y:
    print(f"O primeiro valor ({x}) é maior que o segundo ({y}).")

print("Fim do programa.")
Digite o primeiro valor: 10
Digite o segundo valor: 5
O primeiro valor (10) é maior que o segundo (5).
Fim do programa.

3. Condicional composta (if/else)

A estrutura if/else executa um bloco se a condição for verdadeira, e outro bloco (diferente) se for falsa. É uma escolha entre duas alternativas.

3.1 Sintaxe

if condicao:
    # bloco executado se condicao for True
    instrucao_A
else:
    # bloco executado se condicao for False
    instrucao_B

3.2 Exemplo: par ou ímpar

numero = int(input("Digite um número inteiro: "))

if numero % 2 == 0:
    print(f"O número {numero} é PAR.")
else:
    print(f"O número {numero} é ÍMPAR.")
Digite um número inteiro: 7
O número 7 é ÍMPAR.
💡 Dica de desempenho: Sempre que possível, prefira if/else em vez de dois ifs separados. O if/else executa apenas um teste lógico; dois ifs executam dois testes, dobrando o trabalho do processador.

4. Expressões lógicas (álgebra booleana)

Muitas vezes, precisamos combinar múltiplas condições. Para isso, usamos os operadores lógicos: and (E), or (OU) e not (NÃO).

4.1 Operador and (E)

Resulta em True somente se ambas as condições forem verdadeiras.

ABA and B
False么False么False
False么True么False
True么False么False
True么True么True

4.2 Operador or (OU)

Resulta em True se pelo menos uma das condições for verdadeira.

ABA or B
False么False么False
False么True么True
True么False么True
True么True么True

4.3 Operador not (NÃO)

Inverte o valor lógico: not True = False, not False = True.

4.4 Exemplos práticos

# and: ambas as condições devem ser verdadeiras
idade = 25
tem_carteira = True

if idade >= 18 and tem_carteira:
    print("Pode dirigir.")
else:
    print("Não pode dirigir.")

# or: pelo menos uma condição verdadeira
dia = "sabado"
if dia == "sabado" or dia == "domingo":
    print("Fim de semana!")
else:
    print("Dia útil.")

# not: inverte a condição
chovendo = False
if not chovendo:
    print("Não está chovendo. Pode sair sem guarda-chuva.")

4.5 Precedência dos operadores

Ordem de avaliação (do mais prioritário ao menos prioritário):

  1. Parênteses ()
  2. Operadores aritméticos (**, *, /, %, +, -)
  3. Operadores relacionais (==, !=, >, <, >=, <=)
  4. not
  5. and
  6. or

5. Condicionais aninhadas

Quando temos uma estrutura condicional dentro de outra, chamamos de aninhamento. Isso é útil quando uma decisão depende de outra decisão anterior.

5.1 Exemplo: bônus por tempo de empresa

Funcionários com mais de 10 anos recebem 30% de bônus; com mais de 5 anos (mas menos de 10), 20%; outros, 10%.

salario = float(input("Salário: R$ "))
anos = int(input("Tempo de empresa (anos): "))

if anos > 10:
    bonus = salario * 0.30
else:
    if anos > 5:
        bonus = salario * 0.20
    else:
        bonus = salario * 0.10

print(f"Bônus: R$ {bonus:.2f}")
print(f"Salário final: R$ {salario + bonus:.2f}")
📌 Indentação: Note que o segundo if/else está indentado dentro do else do primeiro. Em Python, a indentação é fundamental para definir a hierarquia dos blocos.

6. Condicional de múltipla escolha (elif)

Quando temos múltiplas condições que dependem da mesma variável, usar if/elif/else é mais organizado do que aninhar vários ifs. O elif é a junção de else + if.

6.1 Sintaxe

if condicao_1:
    # bloco 1
elif condicao_2:
    # bloco 2
elif condicao_3:
    # bloco 3
else:
    # bloco padrão (opcional)

6.2 Exemplo: calculadora com menu

print("=== CALCULADORA ===")
print("1 - Adição")
print("2 - Subtração")
print("3 - Multiplicação")
print("4 - Divisão")

opcao = int(input("Escolha a operação: "))
a = float(input("Primeiro número: "))
b = float(input("Segundo número: "))

if opcao == 1:
    print(f"{a} + {b} = {a + b}")
elif opcao == 2:
    print(f"{a} - {b} = {a - b}")
elif opcao == 3:
    print(f"{a} × {b} = {a * b}")
elif opcao == 4:
    if b != 0:
        print(f"{a} ÷ {b} = {a / b}")
    else:
        print("Erro: divisão por zero!")
else:
    print("Opção inválida!")

6.3 Exemplo: conversão de notas em conceitos

nota = float(input("Digite a nota (0 a 100): "))

if nota >= 90:

elif nota >= 80:

Resumo da apostila

  • if simples: executa um bloco apenas se a condição for verdadeira.
  • if/else: escolhe entre dois blocos (verdadeiro/falso).
  • Operadores lógicos: and (E), or (OU), not (NÃO).
  • Condicionais aninhadas: um if dentro de outro – útil para decisões em cascata.
  • elif: múltipla escolha – mais legível que aninhamentos profundos.
  • Indentação: fundamental em Python – define quais instruções pertencem a cada bloco.
  • Precedência: parênteses > aritméticos > relacionais > not > and > or.
